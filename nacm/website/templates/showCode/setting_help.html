Static Routing

fill with static routing configuration syntax using the python format
example for cisco :

'configure terminal'
'ip route {} {} {}'.format(destination,netmask,gateway)

parameter list available :
  • destination
  • netmask
  • prefix
  • gateway

Dynamic Routing

fill with dynamic routing configuration syntax using the python format
example for cisco :

'configure terminal'
'router ospf {}'.format(id_ospf)
'network {} {} area {}'.format(network,wildcard,area)
'router-id {}'.format(router_id)

parameter list available :
  • id_ospf
  • router_id
  • interface_ospf
  • interface_ripv1
  • interface_ripv2
  • network
  • prefix
  • netmask
  • wildcard
  • area

BGP Routing

fill with BGP routing configuration syntax using the python format
example for cisco :

'configure terminal'
'router bgp {}'.format(asn)
'neighbor {} remote-as {}'.format(neighbor_address,neighbor_asn)
'network {} mask {}'.format(network,netmask)

parameter list available :
  • bgp_name
  • asn
  • router_id
  • neighbor_address
  • neighbor_asn
  • network
  • prefix
  • netmask
  • wildcard
